 Attendance Management System

The Attendance Management System is a web-based application designed to simplify the process of recording and managing student attendance in educational institutions. It provides a secure, efficient, and user-friendly platform for administrators, lecturers, and students to perform attendance-related tasks with ease.

The system consists of three user roles: Admin, Lecturer, and Student. Administrators can manage students, lecturers, subjects, and classes while monitoring attendance reports. Lecturers can create attendance sessions, generate unique session codes, view attendance records, and edit attendance when necessary. Students can mark their attendance by entering the session code, view their attendance history, and monitor their attendance percentage throughout the semester.

This project is developed using HTML, CSS, JavaScript, and Bootstrap for the frontend, Java with Spring Boot for the backend, and MySQL as the database management system. Git and GitHub are used for version control and collaborative development.

The primary objective of this project is to replace traditional paper-based attendance systems with a digital solution that improves accuracy, reduces manual work, and provides quick access to attendance records. The system is designed as an academic project and can be further enhanced with additional features such as QR code attendance, email notifications, and attendance analytics in the future.
